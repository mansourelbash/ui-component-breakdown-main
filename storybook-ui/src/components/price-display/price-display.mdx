# PriceDisplay

Renders a formatted price with currency symbol, amount, and a small note beneath (e.g. "Including Tax"). Used in the product detail page to display the running cart total.

## Usage

```html
<link rel="stylesheet" href="tokens.css" />
<link rel="stylesheet" href="price-display.css" />

<div class="price-display">
  <span class="price-display__currency">JD</span>
  <span class="price-display__amount">85.96</span>
  <span class="price-display__note">Including Tax</span>
</div>
```

No JavaScript is required — this is a purely presentational component.
